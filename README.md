# Fraud Detection System (Real-Time ML Pipeline)

## 🚀 Overview
A production-style ML pipeline that detects fraudulent transactions in real time using machine learning and modern MLOps tools.

## 🧠 Features
- API-based data ingestion
- Data validation & preprocessing
- Feature engineering
- ML models (Random Forest, XGBoost)
- Pipeline orchestration with Airflow
- Experiment tracking with MLflow
- Real-time inference using FastAPI
- Dockerized deployment

## 🛠️ Tech Stack
- Python
- Scikit-learn, XGBoost
- MLflow
- Apache Airflow
- FastAPI
- Docker
